# 🇫🇷 Mettre Ebonhold en français — Guide pas-à-pas

Suis les étapes dans l'ordre. Pas besoin de t'y connaître.

Il y a **deux façons** de faire, choisis la tienne :

- **A. TOUT en français** (menus, quêtes, sorts, echoes…) → il faut télécharger un pack en plus (~2,4 Go).
- **B. Juste les sorts/echoes en français**, le reste du jeu en anglais → plus rapide, pas de gros téléchargement.

---

## 🅰️ Option A — TOUT en français

### Étape 1 : Télécharger le pack de langue française

1. Clique ici : **https://drive.google.com/file/d/1j3OuTz1KMUsuUWXQiMJaE0yQXmAobRxu/view**
2. En haut à droite, clique sur la flèche **Télécharger** (le fichier `WOTLK frFR.rar`, ~2,4 Go).
   *(Google peut dire « impossible d'analyser ce fichier » à cause de sa taille → clique quand même sur « Télécharger ».)*

### Étape 2 : Décompresser le pack

1. Tu obtiens un fichier **`WOTLK frFR.rar`**.
2. Pour l'ouvrir, il faut **7-Zip** (gratuit) si tu n'as pas WinRAR : https://www.7-zip.org → télécharge, installe.
3. **Clic droit** sur `WOTLK frFR.rar` → **7-Zip** → **Extraire ici**.
4. Tu obtiens un dossier **`frFR`**.

### Étape 3 : Placer le dossier `frFR`

1. Ouvre ton dossier **Ebonhold** (celui qui contient `Wow.exe`).
2. Entre dans le dossier **`Data`**.
3. **Copie/colle le dossier `frFR`** dedans.
   *(Au final tu dois avoir : `…\Ebonhold\Data\frFR\`)*

### Étape 4 : Lancer l'installateur

1. Télécharge **`EbonholdFR-Installer.exe`** :
   **https://github.com/Squall98/EbonholdFR-Client/releases**
   (dans la dernière version, le fichier `EbonholdFR-Installer.exe`)
2. **Ferme complètement le jeu.**
3. Double-clique sur **`EbonholdFR-Installer.exe`**.
   - Si Windows affiche un écran bleu **« Windows a protégé votre ordinateur »** :
     clique sur **« Informations complémentaires »** puis **« Exécuter quand même »**. *(C'est normal, le programme n'est pas signé.)*
4. Le dossier Ebonhold est souvent **détecté tout seul** ; sinon clique **Parcourir** et sélectionne-le.
5. Clique sur le bouton **« Tout en français »** (ou règle chaque catégorie à la main : Jeu, Voix, Sorts, Réputations).
6. Clique sur **Appliquer** et attends le message **Terminé**.

> 💡 Tu peux mélanger : par ex. **Jeu = Français** mais **Voix = Anglais** pour avoir le texte en français et les doublages en anglais.

### Étape 5 : Jouer

Lance le jeu normalement. **C'est en français !** 🎉

## 🅱️ Option B — Juste les sorts/echoes en français (sans le gros pack)

1. Télécharge **`EbonholdFR-Installer.exe`** : **https://github.com/Squall98/EbonholdFR-Client/releases**
2. **Ferme le jeu**, double-clique sur l'exe (si écran bleu : *Informations complémentaires → Exécuter quand même*).
3. Sélectionne ton dossier Ebonhold (souvent détecté tout seul).
4. Règle : **Jeu = Anglais**, **Sorts = Français** (et Réputations = Français si tu veux).
5. **Appliquer** → lance le jeu. Les sorts, echoes et factions sont en français, le reste en anglais.

---

## ❓ Problèmes courants

**« Windows a protégé votre ordinateur » (écran bleu)**
→ Normal (programme non signé). Clique **Informations complémentaires → Exécuter quand même**.

**Une erreur dit que le jeu est verrouillé / l'installateur plante**
→ Le **jeu doit être fermé** pendant l'installation. Ferme-le complètement, puis réessaie.

**En jeu : « Your game installation is not up to date » (patch-5/6)**
→ Ton jeu n'est pas à jour (rien à voir avec le français). Lance le **launcher officiel** d'Ebonhold pour mettre à jour, connecte-toi une fois, puis **relance l'installateur**.

**Je veux revenir en anglais**
→ Relance l'installateur, choisis le profil **« Anglais »**, Appliquer. (Tout est réversible.)

**Après une mise à jour du serveur, le français a disparu ?**
→ Relance simplement l'installateur et clique **Appliquer**. C'est rapide.
